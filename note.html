Eternal because every ending is a new beginning, infinite because everything circles back to where it started, perfect because it moves indivisibly as one.

We're all conjured from it but not separate from it. It gives birth to us, raises us, gives birth but doesn't play favourites, influences but doesn't interfere, parents but doesn't punish, gets it done but doesn't get credit.

It's a lot like water. Water helps everyone, but it's not proud, it takes people's shit. It's consistent, open, patient, gracious, fair, useful and adaptable, and above all, it's not proud, so no one resents it.

Be like water.
